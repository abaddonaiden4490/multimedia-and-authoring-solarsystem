HOW TO SET UP THE PROJECT!!!

1. paste the attached code (python code) under "Scripting"

2. put the textures in a folder

3. replace the directory of texture_dir with the location of your textures folder like this:

texture_dir = r"path\to\your\textures\folder"

4. run script
